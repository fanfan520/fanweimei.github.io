2016年金山WPS实习生前端笔试题

第一题：判断一个数字是否是NaN

实现方法如下：首先NaN包括0/0，负数开根号等

方法一：

function pdNaN(num){ return isNaN(num); }

方法二：NaN与任何值都不相等

function pdNaN2(num){ return 2*num==2*num; }

测试：alert(pdNaN2(4)) //输出true

第二题：sum(2,3)=>5;sum(2)(3)=>5;用代码实现她们

sum(2,3)的实现：

function sum(a,b){ if(isNaN(a)&&isNaN(b)){ return a+b; }else{ alert('a或b不是数值') } }

sum(2)(3)的实现：

function sum(a){ return function(b){ return a+b; } }

测试：alert(sum(2)(3)); //输出5

扩展 实现sum(1)(2)(3)(4)

使用了闭包的知识

function sumt(a){ var sum=a; var tmp=function(b){ sum=sum+b; return tmp; } tmp.toString=function(){ return sum; } return tmp; }

测试：alert(sumt(1)(2)(3)(4)(5)); //输出15
